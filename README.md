# FinSight AI

FinSight AI is an AI-powered financial market intelligence platform designed to analyze financial and stock market data using machine learning.

## Project Goals

- Collect and process financial market data
- Store structured financial data in MySQL
- Perform exploratory data analysis
- Engineer financial and technical features
- Build and evaluate machine learning models
- Apply explainable AI techniques
- Develop an interactive Streamlit application
- Create executive dashboards using Power BI

## Tech Stack

- Python
- MySQL
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- SHAP
- Plotly
- Streamlit
- Power BI
- Git & GitHub
- Docker
